# Altur Frontend

React/Vite frontend scaffold managed with `bun`.

## Setup

Install dependencies:

```sh
bun install
```

Configure the API base URL:

```sh
cp .env.example .env
```

Update `VITE_API_BASE_URL` in `.env` for the target backend.

## Scripts

```sh
bun run dev
bun run build
bun run lint
bun run typecheck
```
